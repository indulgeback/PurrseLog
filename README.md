## Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- A device or emulator to run the app

### Installation

1. Clone the repository:

```bash
git clone git@github.com:indulgeback/PurrseLog.git
cd PurrseLog
```

2. Install dependencies:

```bash
flutter pub get
```

3. Run the app:

```bash
flutter run
```
